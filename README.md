# Nixx — Portfolio

Portofolio pribadi, single-page, dark cosmic theme. Dibangun pakai HTML/CSS murni tanpa framework, siap deploy ke Vercel.

## Isi Project

```
.
├── nixx-portfolio.html   # halaman utama portofolio
├── vercel.json           # konfigurasi deploy Vercel
└── README.md
```

## Menjalankan Secara Lokal

Cukup buka `nixx-portfolio.html` langsung di browser, tidak butuh server atau build step.

## Deploy ke Vercel

1. Push repo ini ke GitHub.
2. Buka [vercel.com/new](https://vercel.com/new) dan import repo.
3. Framework preset pilih **Other**, biarkan build command kosong.
4. Deploy — halaman otomatis kebuka di root URL sesuai `vercel.json`.

Atau lewat Vercel CLI:

```bash
npm i -g vercel
vercel --prod
```
